Company Description

Antero Software builds workflow systems to help organisations scale while remaining customer-focused, without creating manpower overhead. We have developed a range of software offerings for industries including Financial Services, Medical, Biotech, Education, and Construction. Our office is based in Brisbane, QLD, but we offer flexibility for remote work.

Role Description

This is a full-time hybrid role for a Software Engineer located in Brisbane, QLD, with flexibility for some remote work. The Software Engineer will be responsible for day-to-day tasks associated with software development and back-end web development.

Duties

  • researching, consulting, analysing and evaluating system program needs
  • identifying technology limitations and deficiencies in existing systems and associated processes, procedures and methods
  • testing, debugging, diagnosing and correcting errors and faults in an application
  • within established testing protocols, guidelines and quality standards to ensure programs and applications perform to specification
  • writing and maintaining program code to meet system requirements, system designs and technical specifications in accordance with quality accredited standards
  • writing, updating and maintaining technical program, end user documentation and operational procedures
  • providing advice, guidance and expertise in developing proposals and strategies for software design activities such as financial evaluation and costing for recommending software purchases and upgrades
  • managing testing and automation of software and application deployments
  • collecting and analysing operational metrics
  • developing continuous improvement and continuous delivery strategies across system design and software development
  • the creation and development of the tools required to support software and its management and security
  • performing forensic analysis to identify anomalies or threats
  • encryption and decryption
  • identification and mitigation of risks that may affect the performance and security throughout the lifecycle of the product.

Qualifications

  • Computer Science and Software Development skills
  • Experience in Back-End Web Development
  • Strong programming skills and knowledge of Object-Oriented Programming (OOP)
  • Excellent analytical and problem-solving skills
  • Experience with programming languages like Java, Python, or Ruby on Rails
  • Experience with cloud-based technologies like Amazon Web Services (AWS) or Microsoft Azure is a plus
  • Bachelor's or master's degree in Computer Science or related field is not required, but encouraged

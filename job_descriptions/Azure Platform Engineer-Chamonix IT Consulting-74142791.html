What will you be doing?

As an Azure Platform Engineer with Chamonix, you will play a crucial role in designing, constructing and maintaining Azure environments for our clients' mission-critical systems. You will work closely with solution architects, project teams, and technical stakeholders to develop best practices and standards that align with business objectives while considering performance, scalability, security, and cost-efficiency.

How will you help contribute to our success?

  • Design, deploy, and manage Azure infrastructure components, including virtual machines, virtual networks, storage, and Entra ID services.
  • Create and manage Azure resources using Infrastructure as Code (IaC) tools such as Azure Resource Manager templates, Bicep, or Terraform.
  • Create and manage pipelines and deployments for Infrastructure as Code (IaC) using tools such as Azure DevOps or GitLab.
  • Implement monitoring solutions and perform regular optimization of Azure resources to ensure cost-efficiency and high availability.
  • Implement security best practices and compliance policies in Azure, including identity and access management, network security, and data encryption.
  • Develop and maintain automation scripts and workflows to streamline Azure provisioning and management tasks.
  • Set up and manage Azure-based backup and disaster recovery solutions to ensure data protection and business continuity.

What skills does our ideal candidate have?

  • Experience in designing, deploying, and managing Azure solutions, including infrastructure, networking, and security components.
  • Proficiency in scripting languages (e.g., PowerShell, Python) and experience with automation tools (e.g., Terraform, Ansible).
  • Familiarity with IaC tools and practices, such as ARM templates, Azure DevOps, Terraform, or Ansible.
  • Strong understanding of Azure networking concepts, including virtual networks, private link, peering, VPNs, and ExpressRoute.
  • Knowledge of Azure security best practices, including identity and access management, network security groups, and Azure Security Center.
  • Experience with Azure monitoring and logging solutions, such as Azure Monitor and Log Analytics.
  • Proficiency in Azure backup and disaster recovery solutions, including Azure Backup and Azure Site Recovery.
  • Understanding of managed container orchestration platforms like Kubernetes in Azure Kubernetes Service (AKS).

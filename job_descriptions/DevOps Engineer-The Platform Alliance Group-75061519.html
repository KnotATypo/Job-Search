About the Role
We are searching for a talented DevOps Engineer to join our growing team. In this role, you will be responsible for designing, building, and maintaining our infrastructure using Infrastructure as Code (IaC) principles and tools. You will collaborate closely with developers and operations teams to ensure a smooth and efficient software development lifecycle.

Responsibilities
  • Automate infrastructure provisioning and configuration management using tools like Terraform, Ansible, or Chef.
  • Implement and manage continuous integration/continuous delivery (CI/CD) pipelines using tools like Jenkins, GitLab, or CircleCI.
  • Configure and manage cloud infrastructure on AWS
  • Monitor and troubleshoot infrastructure issues to ensure high availability and performance.
  • Develop and implement security best practices for infrastructure and application deployment.
  • Stay up-to-date with the latest DevOps tools and technologies.

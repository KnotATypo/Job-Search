Oracle & SQL Server Database Administrator

  • 4 month contract with possible 1 x 12 month extension
  • Carseldine location

As the Oracle & SQL Server Database Administrator, your key responsibilities will include:

  • SQL Server DBA - SQL Clustering, backup and recovery, monitoring, maintenance, SSRS, SSIS and Patching
  • AWS infrastructure - SQL Server, other DB types (Postgres/Aurora), supporting infrastructure, EC2, RDS, FSx and S3
  • Azure Infrastructure - SQL Server and Azure managed databases
  • Preferred (But not mandatory)
  • Oracle on Exadata/Data Guard/Clustering/Containers/OCI
  • Working with Outsource Service Provider
  • Agile tools such as Jira\Confluence.
  • Resolve complex problems involving database products.
  • Assist in the design and implementation of enterprise database enterprise monitoring and management products to ensure service availability.
  • Assist in the design and implementation of Database Backup, High Availability and Disaster Recovery technologies
  • Advise and provide recommendations and solutions to the Principal Team Leader (Database Services) regarding future direction, technical support and issues resolution for Database related infrastructure and services.
  • Develop & enhance procedures and processes within the Database Services team to improve the level of service provided to clients.
  • Contribute to the development of policies within the database environments.
  • Maintain knowledge of the technical interdependencies within the database environments.
  • Provide advice and guidance to software maintenance and project teams on data modelling and best practise use of database products.
  • Provide advice, effective mentoring and skills transfer activities that contribute to a high-performance Database Services team.
  • Perform release and change management activities on behalf of the Database Services team.
  • Provide database management skills for operational and business projects as required.

Current and varied experience across public and private sector is highly desirable.

Software Engineer

Job Responsibilities:

  • Responsible for back-end technology implementation scheme, application and business system optimization and integration
  • Solve the technical problems encountered in software development and testing to ensure software reliability
  • Responsible for daily project version upgrade, deployment and implementation
  • Prepare software development design documents according to the requirements of the company‘s software development specifications to ensure the transparency and controllability of the development process.
  • Write unit/functional or integration tests and perform unit testing
  • Code review for other team members.
  • Automate the maintenance work with the development tools.

  • Bachelor degree or above in Computer Science / engineering related with experience in large-scale projects;
  • Over 5 years software development experience.Proficient in Java,can skillfully use Spring boot, Spring MVC, Spring Cloud and other open source frameworks;
  • Familiar with the design and application of distributed system, and familiar with distributed, cache, message, search and other mechanisms;
  • Able to reasonably apply common distributed technologies and solve problems;
  • Proficient in Mysql, postgresql, Redis and other databases, familiar with database performance optimization;
  • Master Linux operating system, skillfully use a script language like shell, be able to independently complete system deployment and daily maintenance of back-end applications, and have rich experience in SQL optimization;
  • Skillfully use eclipse, idea and other development tools, and be familiar with Git, Maven and other project management tools;
  • Familiar with CI/CD, devops and kubernetes related tools and technologies;
  • Front End development experience with reactjs or vuejs will be a plus;
  • Good communication skills both in English and able to work under pressure

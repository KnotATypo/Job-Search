We are looking for an enthusiastic, self-motivated team player with excellent technical and communication skills, that will be able to quickly understand and adhere to organisational policy, procedures, and security requirements for supporting and maintaining our Identity, Authentication and Network Addressing environments. The purpose of this role is to provide operational support within the Enterprise Active Directory team supporting the key technologies listed below.

Required Skills

  • Active Directory Infrastructure installation, maintenance, and troubleshooting.
  • Scripting and automation (PowerShell, DSC, etc.)
  • Microsoft Windows Server technologies
  • Network Addressing infrastructure (DNS, DHCP)
  • Service Now or equivalent ITSM Tool
  • ITIL Framework and associated ITSM standards

Nice to Have Skills

  • Active Directory Federation Services
  • Enterprise PKI Infrastructure
  • Entra ID Administration
  • VMWare ESX
  • Microsoft SCOM

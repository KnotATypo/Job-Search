Role:
We are looking for a Mainframe Application Tester- you will be involved in Application and Batch Process Testing for functional and non-functional requirements.
  • Use ISTQB test design techniques to develop test plans and approaches.
  • Create and maintain test cases and test suites from analysis of functional and non-functional specifications.
  • Identify test data and environment requirements.
  • Produce test traceability from specifications to test cases and record test outcomes.
  • Capture test evidence to validate test execution activities.
  • Report and action incidents/defects as per agreed defect management standards.
  • Clear communication of testing progress and results with stakeholders.
  • Assist Test Lead/Project Manager with test completion/summary report/s.
  • Provide technical and testing advice and support during user acceptance phases

  • Mainframe batch jobs OR Control-M GUI – Java Batch
  • IBM WebSphere Application Server (WAS) Middleware,
  • Web Services (XML/SOAP) and some Integration Platform API testing (MuleSoft);
  • Oracle databases;

  • Log Checking
  • JMeter / SoapUI; /ReadyAPI
  • JIRA and Confluence;

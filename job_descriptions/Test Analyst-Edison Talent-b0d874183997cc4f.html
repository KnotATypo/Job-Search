The Opportunity:
With multiple programs of work on the go, this State Government Department is seeking talented and professional Testers to implement and support a number of test phases within these programs and related projects.

We are currently seeking a Test Analyst, with a potential Senior Test Analyst to be released shortly.

Please note: Candidates must be Brisbane based or willing to relocate to Brisbane for these opportunities.

Key Responsibilities:

  • Develop and assist with the development of Test Plans and testing documentation.
  • Conduct peer reviews of test cases for accuracy
  • End-to-end manual testing
  • Application testing
  • Using various defect management tools
  • Sound experience in test delivery and using various testing approaches including Agile and TDD

Suitable Candidates Will Possess:

  • Experienced in software and technical testing
  • Able to develop test scenarios & UAT
  • API Testing
  • Postman
  • Any experience with Automation Testing (Selenium, Cucumber framework/BDD, Jenkins CI/CD) would be viewed favourably but not essential
  • Coding proficiency in Java or C# or other languages would be an advantage but not essential
  • JIRA, Confluence or Zephyr
  • Ideally possess SQL query writing ability
  • Working knowledge and experience in various SDLC approaches including Agile, Waterfall and Hybrid
  • Excellent analytical and problem-solving skills
  • Computer Science / IT related degree
  • ISTQB Foundations certificate is ideal but not essential
  • Strong interpersonal skills including solid written, and oral communication ability

Benefits: Joining our client offers you:

  • Competitive salary and remuneration
  • Work life balance and flexibility in working arrangements
  • A collaborative and inclusive working culture
  • A focus on continuous learning and growth

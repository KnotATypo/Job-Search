About the Role

You’ll join the team as a Cloud Engineer to provide Level 1/2 cloud support, focusing on deployment, maintenance, and issue resolution for our customers in the Emergency Services industry.

You’ll have the opportunity to:

  • Provide primary support for AWS environments, proactively identifying and troubleshooting issues, ensuring the continuous functionality and optimization of AWS infrastructure.
  • Oversee Windows environments, including effective server management for optimal performance and reliability.
  • Conduct regular patching activities to ensure systems are up to date with the latest security and performance updates.
  • Maintain comprehensive documentation of environment configurations, issue resolutions, and best practices.
  • Continuously assess and optimize AWS infrastructure for performance, scalability, and cost-effectiveness.
  • Perform limited hardware maintenance to address any critical issues and maintain system integrity.
  • Work with external cloud partners to improve the overall quality of cloud services.

What you’ll bring to the role:

  • Bachelor’s degree in Computer Science, IT, or equivalent.
  • Experience supporting AWS environments as part of a Level 1/2 service desk or managed service.
  • Experience with infrastructure as code in AWS environments.
  • Familiarity with Windows environments, including server management, patching, and basic hardware maintenance.
  • Familiarity with security tools and practices for cloud environments.
  • Experience with version control systems such as Git for managing infrastructure changes.
  • Proactive attitude and adaptability, especially in handling diverse tasks beyond AWS, reflecting the changing nature of the role.
  • AWS certification; ideally including Cloud Practitioner.

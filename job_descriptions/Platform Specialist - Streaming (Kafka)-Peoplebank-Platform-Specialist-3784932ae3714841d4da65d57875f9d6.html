  • Technical Writing and analytical skills highly desirable

  • Familiarity with Amazon MSK and Confluent Cloud
  • Familiarity with AWS networking, infrastructure, ways of working, security, capacity planning
  • Familiarity with API platforms including the design, development and implementation of Rest APIs with OpenAPI specification. Knowledge of Azure APIM or AWS Gateways is essential
  • Experience in projects that have conducted the Agile practices including Sprint Planning, Backlog Refinement and Daily Standups
  • Experience with CI/CD DevOps processes using Terraform and Github related products.

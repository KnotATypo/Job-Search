The Integration Engineer will work together with teams across to:

• design, implement, and maintain seamless integration solutions within our technology ecosystem
• be responsible for collaborating with cross-functional teams, leveraging technical expertise in C#, microservices, and Azure to enhance system connectivity and optimize data flow
• be passionate about integration technologies, enjoy problem-solving, and have a strong desire to learn and grow in a collaborative environment

Experience - Mandatory
• C#: Proficiency in C# programming language for developing robust and efficient integration solutions.
• Microservices Knowledge: Understanding and practical experience with microservices architecture.
• Azure Experience: Hands-on experience with Azure services, including but not limited to Azure Logic Apps, Azure Service Bus, and Azure Functions.
• Critical Thinking: Strong analytical and problem-solving skills to address complex integration challenges.
• Debugging with Azure: Ability to effectively debug and troubleshoot integration issues using Azure tools and resources.
• Strong Team engagement, ability to influence and lead the change journey

Experience – Nice to have
• Azure Functions: Familiarity with serverless computing and the ability to leverage Azure Functions in integration scenarios.
• API Management: Knowledge of Azure API Management to create, publish, and manage APIs effectively.
• Domain Driven Design (DDD): Understanding of DDD principles and their application in designing integrations aligned with business domains.
• Infrastructure as Code (IaC): Experience with IaC tools such as Terraform or Azure Resource Manager templates

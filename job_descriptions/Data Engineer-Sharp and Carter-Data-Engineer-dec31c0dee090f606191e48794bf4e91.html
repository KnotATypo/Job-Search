The Role: Data Engineer
6 Months Fixed Term Contract (FTC) / Potential option to go permanent
Location: Fortitude Valley

Sharp and Carter are excited to be supporting a fantastic non-profit organisation in their search for a Data Engineer for a fixed-term contract of six (6) months. This organisation is well-known for its impactful work improving the lives of Queenslanders in need. So, if you're a Data Engineer looking to contribute to a meaningful cause and be proud of the work that you do, this opportunity might just be what you're looking for!

Day-to-Day Responsibilities:
  1. Design, implement, and maintain our clients data warehouse to support analytics and reporting needs.
  2. Collaborate with cross-functional teams to create and document data models and mappings using tools like Power BI, Google Data Studio, and SSAS models.
  3. Implement and maintain security measures, including data security protocols and user access controls.
  4. Enforce data governance policies and procedures to ensure compliance and data quality standards.
  5. Use SQL query and stored procedure development to extract insights from data.
  6. Design and develop ETL processes and data pipelines to automate data ingestion, transformation, and loading tasks.
  7. Monitor database performance metrics and optimise query performance and resource utilization.
  8. Produce architecture diagrams and documentation to communicate data infrastructure designs effectively.
  9. Provide technical mentoring and guidance to less experienced members of the team.
Required Knowledge and Experience:
  • Relational Databases: Advanced experience in SQL Server or other major relational database systems.
  • Data Warehousing and ETL/ELT: Strong understanding of data warehousing principles, ETL/ELT processes, and dimensional data modeling.
  • Database Security and Administration: Demonstrated expertise in implementing and administering database security measures.
  • Microsoft BI Stack: Experience with Microsoft BI tools including SQL Server, SSAS, SSRS, SSIS, and T-SQL.
  • Integration Tools: Ideally, familiarity with integration tools such as Azure Data Factory.
  • Structured and Semi-structured Data: Experience handling both structured and semi-structured data formats like JSON, Cosmos, and XML.
Essential Qualification/s and Experience
  • Tertiary qualifications in Computer Science, Information Technology, or a related field.

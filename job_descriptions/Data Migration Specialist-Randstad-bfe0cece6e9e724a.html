Full job description

Randstad are currently working with a large government department to find an experienced Data Migration Specialist for an initial 3 Month contract with likelihood of extension. This role is based in Brisbane city, however remote and WFH candidates will be considered. The ideal candidate for this role will possess the following:


Role Requirements:

  • Proven experience in SQL Server Integration Services (SSIS) and straight SQL.
  • Minimum of 10 years’ experience as Data migration specialist from legacy SQL system to D365.
  • Experience in developing & implementing ETL solutions using Oracle, Microsoft SQL, and any other Relational Database Management System (RDBMS) databases.
  • Experience in creating and maintaining source to target mapping, transformation rules, loading methodology, error and exception reporting and implementation documents to support the ETL processes required to refresh the target database or data mart via multiple sources.
  • Exceptional SQL Scripting, data management and data analysis skills including the use of SSIS integration tools.
  • Exceptional written and verbal communication skills, with an ability to communicate with both technical and non-technical stakeholders.


Experience in:

  • Data Migration from SQL to D365
  • Data Mapping
  • Conceptual data mapping
  • Entity Relationship Diagrams (creation and analysis)
  • Schema review and creation
  • Transformation rule formulation
  • Azure Data Factory set up with min / max batch sizes
  • Azure pipeline heath checks
  • Visual Studio
  • Expert level – problem solving and logical thinking
  • Data Protection acumen
  • ETL framework creation, analysis and implementation


Good understanding of:

  • Advanced understanding of database design and management
  • Experience working with large amounts of data in a variety of formats and structures
  • Expertise in programming languages such as SQL,
  • Strong analytic skills and the ability to recognize patterns or trends in data
  • Understanding of models related to the field, such as:
  • Database design development
  • Segmentation techniques
  • ETL frameworks (planning, set up, execution, error and exception reporting, some automation, progress reporting)
  • Data security and protection in ETL
  • Understand project, program, risk and data governance frameworks.
  • Able to insert themselves into any stage/ phase of a project or program comfortably.

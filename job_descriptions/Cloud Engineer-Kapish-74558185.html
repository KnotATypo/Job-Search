What about the role:

The main function of this position is to provide operational support for our Microsoft Azure based solutions. A typical day for you would involve:

  • Monitoring cloud infrastructure environments.
  • Implementing and maintaining service integrations, frameworks, Infrastructure as Code, CI/CD, monitoring tools, DevOps and cloud services.
  • Monitoring cloud costs and providing input into improvements.
  • Resolution of incidents, diagnosis of problems and completing changes following ITIL best practices.
  • Providing service to customers in support of requirements and service level agreements.
  • Development and execution of scripts to support environments.
  • Contributing towards knowledge management and updating the CMDB.
  • Performing preventative maintenance across Windows environments.
  • Automating tasks and adding towards continuous improvement.

What requirements do I need:

  • Must be Australian citizen . NV1 and/ or baseline security clearance highly desirable.
  • Understanding of DevOps principles, cloud computing technologies and business drivers.
  • Advanced knowledge of cloud-based infrastructure and networking concepts as well as experience with hybrid environments.
  • Exposure to technologies such as Ansible, TerraForm, CloudFormation, Azure Resource Manager, Kubernetes, Jenkins, BitBucket, Git and other CICD solutions.
  • Skill with scripting languages such as PowerShell (preferred), Bicep or Python.
  • Linux administration experience.
  • Basic experience with configuration of the security stack (firewalls, VPC, AV, IDS/IPS, DLP).
  • Working knowledge of security frameworks such as ISO27001, IRAP, SOC or equivalent.
  • Experience with Service Management tools such as Confluence, JIRA.
  • Strong capability with Logging/Monitoring tools such as Elastic, NewRelic and Splunk.

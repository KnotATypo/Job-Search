NTT Brisbane is currently seeking a skilled Cloud Security Engineer to join our dynamic team. In this role, you will play a pivotal role in designing, implementing, and maintaining secure solutions for our cloud environments. Your primary focus will be ensuring the security of our cloud applications while collaborating closely with our applications team.

Key Qualifications:
Proficiency in cloud and SaaS security concepts, techniques, and tools, encompassing cloud computing, architecture, security, and services.
Familiarity with common cloud security vulnerabilities and threats and standard security frameworks is a must.
Proficiency in programming languages (Python, JavaScript, Java, or C#) is a must.
Hands-on experience with cloud environment.

Key Responsibilities:
Develop and implement secure coding practices, security standards, and guidelines tailored for cloud application environment within the NTT Brisbane framework.
Conduct thorough security reviews, audits, and testing of all cloud application products and code to ensure compliance and identify potential vulnerabilities.
Monitor and respond promptly to security incidents and breaches, implementing necessary measures to minimize impact and ensure asset integrity.
